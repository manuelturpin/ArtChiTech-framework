### Grille Tarifaire

| Édition | Prix | BOM | Marge Brute | Volume cible |
|---------|:----:|:---:|:-----------:|:------------:|
| **Standard** | $899 | $100 | 89% | 2,000/an |
| **Premium** (titane) | $1,299 | $150 | 88% | 500/an |
| **Collector** (édition limitée) | $1,999 | $200 | 90% | 100/an |

### Justification Premium Pricing

1. **Coût perçu ≠ Coût réel** - Pricing basé sur la valeur, pas sur le BOM
2. **Exclusivité** - Éditions limitées numérotées
3. **Segment cible** - Early adopters et collectionneurs tolerant au prix
4. **Positionnement** - Concurrent des montres hybrides luxe ($800-$2000)

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    DISTRIBUTION STRATEGY                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │

│  ─────────────────────────                                             │

│                                                                         │
│  PHASE 2: SCALE (Mois 7-18)                                            │
│  ─────────────────────────                                             │
│  ├── Site D2C                    [50%]  Core channel                   │
│  ├── Amazon Premium              [20%]  Reach, découvrabilité         │
│  └── Boutiques sélectives        [30%]  Horlogeries indépendantes     │
│                                                                         │
│  PHASE 3: EXPANSION (Mois 19+)                                         │
│  ──────────────────────────                                            │
│  ├── Site D2C                    [40%]  Brand flagship                 │
│  ├── Retail premium              [30%]  Department stores luxe        │
│  ├── Horlogeries                 [20%]  Network spécialisé            │
│  └── B2B/Partnerships            [10%]  Corporate, hospitality        │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### Détail des Canaux

| Canal | Avantages | Inconvénients | Marge nette |
|-------|-----------|---------------|:-----------:|
| **D2C (site)** | Contrôle total, données client | Acquisition coûteuse | 75-85% |
| **Kickstarter** | Validation, PR, cash upfront | Fees 8-10%, risque | 65-70% |
| **Amazon Premium** | Reach massive | Fees 15%, commoditization | 55-65% |
| **Horlogeries** | Crédibilité, conseil | Wholesale -50% | 35-45% |
| **Department stores** | Volume, visibilité | Conditions dures | 30-40% |

### Recommandation: D2C-First

**Justification:**
1. **Marges maximales** - Critique pour un nouveau produit
2. **Relation directe** - Feedback loop rapide
3. **Brand control** - Éviter la dilution
4. **Data ownership** - Email, comportement, LTV

### Break-Even Analysis

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    BREAK-EVEN ANALYSIS                                 │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  Fixed Costs (Monthly):                                                │
│  ├── Team (3 FTE)                 $30,000                              │
│  ├── Marketing base               $10,000                              │
│  ├── Operations                   $8,000                               │
│  └── G&A                          $7,000                               │
│  TOTAL FIXED                      $55,000/month                        │
│                                                                         │
│  Variable Costs (per unit):                                            │
│  ├── COGS                         $120                                 │
│  ├── Shipping                     $15                                  │
│  ├── Payment processing           $30                                  │
│  └── CAC (avg)                    $80                                  │
│  TOTAL VARIABLE                   $245/unit                            │
│                                                                         │
│  Contribution Margin:                                                   │
│  Revenue per unit                 $1,000                               │
│  - Variable costs                 $245                                 │
│  = CONTRIBUTION                   $755/unit                            │
│                                                                         │
│  Break-Even (units/month):                                             │
│  $55,000 ÷ $755 = ~73 units/month                                     │
│                                                                         │
│  Break-Even (annual):             ~880 units                           │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 9. Conclusions

### ✅ Viabilité Business Confirmée

1. **Marges excellentes** - >80% gross margin possible
2. **Break-even atteignable** - ~880 unités/an
3. **Multiple revenue streams** - Pas uniquement hardware
4. **Scalable** - D2C puis retail

### ⚠️ Points d'Attention

1. **CAC à surveiller** - Critique pour la rentabilité
2. **Cash flow** - Besoin de financement initial
3. **Validation marché** - Kickstarter comme test

### 📊 Score Business: 7.5/10

**Verdict**: Business model viable avec des marges attractives. Le succès dépend de la validation Kickstarter et du contrôle du CAC.
