# Phase 2 : Stratégie

**Objectif** : Définir la roadmap et le modèle économique

**Skills actifs** : `superpowers:writing-plans`, `superpowers:brainstorming`

## Checklist

- [ ] Business Model Canvas complété
- [ ] Revenue streams identifiés
- [ ] Pricing strategy définie
- [ ] Go-to-Market strategy documentée
- [ ] Roadmap produit (3-6-12 mois)
- [ ] MVP scope défini (must-have vs nice-to-have)
- [ ] Success metrics (KPIs) définis
- [ ] Risques identifiés et mitigation planifiée
- [ ] Resources nécessaires estimées
- [ ] Timeline réaliste établie
- [ ] Stakeholders alignés
- [ ] Go/No-Go documenté

## Anti-Patterns

⚠️ **Feature creep** : Ajouter des features au MVP qui ne sont pas essentielles.

⚠️ **Unrealistic timelines** : Sous-estimer le temps de développement.

⚠️ **Ignoring competition** : Ne pas analyser comment vous différencier.

⚠️ **Vague success metrics** : "Plus d'utilisateurs" au lieu de "1000 DAU en 3 mois".

## Outils Recommandés

- **Notion** : Roadmap, Business Model Canvas
- **Miro** : Strategy workshops
- **Spreadsheet** : Financial projections
- **Jira/Linear** : Backlog prioritization

## Métriques Clés

| Métrique | Cible |
|----------|-------|
| MVP features | 3-5 core |
| Time to MVP | < 3 mois |
| CAC estimé | Défini |
| LTV estimé | > 3× CAC |

## Transition → Phase 3

Conditions Go :
- MVP scope verrouillé
- Business model viable
- Roadmap approuvée par stakeholders
